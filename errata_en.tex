Due to limitations in our abilities, there are inevitably some errors in the book content, for which we apologize. In this document, we list the errors in the book since the first printing (August 2019). Most of these errors should be corrected in the next printing. Therefore, readers should pay attention to the printing times on the title page and compare them with this document.

For changes in the code, please refer to the current GitHub page; this errata is only for book content. This errata is only valid for the second edition of the book; for errata in the first edition, please refer to the corresponding GitHub page.

As typesetting may have minor changes in different printings, the page and paragraph markings in the errata are only valid for the current printing.

If you believe there are errors in the book content, please email me or send a message through the feedback channel provided by the publisher. Sending an email directly to me would be more convenient.

\section{Additional Changes Not Included in the Errata}
In addition to the above corrections, there are some changes not included in the book text, which need to be explained to readers. Most of these errata come from GitHub issues. They have been fixed in the code, but the book text may lag behind the speed of updates on GitHub.

\begin{enumerate}
\item In the example of SE3 construction in Lecture 4, quaternion should be in the order of $w,x,y,z$.
\item In the orb_self.cpp example in Lecture 7, line 375 should be changed to int idx_pq = i * 32 + k;
\item In the orb_self.cpp example in Lecture 7, lines 360 and 361, m01 and m10 should be swapped.
\item For the implementation of direct method in Lecture 8, cv::parallel_for_function supported by OpenCV 4 is required. If readers are using older versions, some modifications need to be made to the code. Please refer to the corresponding version of OpenCV documentation or refer to~\url{https://github.com/gaoxiang12/slambook2/issues/32}.
\end{enumerate}
